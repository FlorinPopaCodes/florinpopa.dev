# Design: Brutalist Portfolio

## Principles

Following [brutalist-web.design](https://brutalist-web.design/):
- Content is readable on all devices
- Only hyperlinks and buttons respond to clicks
- Hyperlinks are underlined, buttons look like buttons
- No decoration that doesn't serve content
- Performance is a feature

## Visual System

**Colors** (matching florinsays.com):
- Background: `#0c0c0c`
- Text: `#c8c4bd` (body), `#e8e4dd` (bright), `#908a84` (muted)
- `color-scheme: dark` on `<html>`

**Typography**:
- Body: Georgia, Times New Roman, serif (19px / 1.75)
- Headings: system-ui, sans-serif
- Metadata: ui-monospace, SF Mono, monospace

**Layout**: 600px max-width, centered, 4rem vertical padding
